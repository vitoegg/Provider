# Provider
个人自用的Loon、Surge模块和规则，以及一些自用的网络脚本

1、Surge模块主要引用以下仓库

- https://surge.qingr.moe
- https://github.com/zirawell/R-Store

2、Loon模块主要引用以下仓库

- https://github.com/luestr/ProxyResource
- https://github.com/linuszlx/Loon

3、规则从各个仓库中搜集，以及自己抓包补充
